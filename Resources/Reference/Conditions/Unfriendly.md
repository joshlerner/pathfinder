# Unfriendly

This condition reflects a creature's disposition toward a particular character, and only supernatural effects can impose this condition on a PC. A creature that is unfriendly to a character dislikes and distrusts that character. The unfriendly creature won't accept [[Request|Requests]] from the character.

**Source** *Pathfinder Player Core pg. 447 <sup>2.0</sup>*
