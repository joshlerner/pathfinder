# Hostile

This condition reflects a creature's disposition toward a particular character, and only supernatural effects can impose on a PC. A creature hostile to a character actively seeks to harm that character. It doesn't necessarily attack, but it won't accept [[Request|Requests]] from the character.

**Source** *Pathfinder Player Core pg. 444 <sup>2.0</sup>*
