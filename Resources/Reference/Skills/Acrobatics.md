# Acrobatics (Dex)

Acrobatics measures your ability to perform tasks requiring coordination and grace. When you use the [[Escape]] basic action, you can use you Acrobatics modifier instead of your unarmed attack modifier. You can also use it for the basic actions [[Arrest a Fall]] and [[Grab an Edge]] instead of Reflex.

**Source** *Pathfinder Player Core pg. 233 <sup>2.0</sup>*
